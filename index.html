Quick Start

1. Add Dependency

The library is hosted on Maven Central, so no additional repository configuration is needed if you already have mavenCentral() in your repositories block.

// Module build.gradle.kts dependencies { implementation("se.premex.compose.preview:android-compose-preview-ext:1.0.0") }

2. Use in Previews

import androidx.compose.ui.tooling.preview.Preview import se.premex.compose.preview.device.catalog.android.Zebra @Preview(name = "Zebra MC33", device = Zebra.MC33) @Preview(name = "Zebra EC50", device = Zebra.EC50) @Composable fun MultiDevicePreview() { YourComposableContent() }
